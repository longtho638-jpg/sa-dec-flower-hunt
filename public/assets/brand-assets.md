# AGRIOS.tech Brand Assets

## Logo Usage

The AGRIOS.tech logo consists of:
- Primary Color: Emerald (#10B981)
- Gradient: Emerald → Cyan (#10B981 → #06B6D4)
- Background: Black (#000000)
- Font: JetBrains Mono for tech feel

## Brand Identity

- **Full Name**: AGRIOS TECHNOLOGY CO., LTD
- **Brand Name**: AGRIOS.tech
- **Tagline**: Agriculture + Operating System
- **Campaign**: Sa Đéc Flower Hunt 2026

## Brand Colors

| Name | Hex | Usage |
|------|-----|-------|
| **Emerald 500** | #10B981 | Primary, CTAs |
| **Cyan 500** | #06B6D4 | Accent, gradients |
| **Stone 950** | #0c0a09 | Background |
| **Amber 500** | #F59E0B | Highlights, alerts |

## Typography

- **Headlines**: Inter, light weight
- **Body**: System fonts
- **Code/Data**: JetBrains Mono

## Logo Component

Import from `@/components/brand/AgriosLogo`

```tsx
import { AgriosLogo } from "@/components/brand/AgriosLogo";

<AgriosLogo className="w-10 h-10" />
```

## Usage Guidelines

1. Maintain clear space around the logo
2. Do not distort or recolor
3. Use on dark backgrounds preferred
4. Always display with `.tech` suffix in text
